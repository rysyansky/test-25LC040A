Detailed Description

Class for working with data on 25LC040A EEPROM.

This class provides reading and writing functionality for 25LC040A microchip by abstract driver SPIBitBanger. SPIBitBanger is considered to be capable of setting CS, WP pins high and low, synchronising communication, reading from SO and writing to SI.

Warning
This class do not provide hyper-extencive error-handling. All passed data is considered to be valid.

Splits 9bit address into 2 required addresses.

Splits 9bit address into two 8bit sequences. Commands READ and WRITE requires sending two addresses as [0000 a8c2c1c0][a7-a0] a8 is 9th bit of 9bit address - [000a8 a7-a0], it is used to address full 512 byte array c2-c0 is command reserved for READ or WRITE command

Parameters
address9bit address
commandWRITE or READ command
commandbytePointer to desired variable for command byte
addressbytePointer to desired variable for 8bit address byte
